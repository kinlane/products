Products API

This is a demo API for making data available about products, it is hosted as a Github repository, and is openly licensed for re-use.

This API is maintained by Kin Lane (@kinlane), the API Evangelist. If you have any questions, comments or there are APIs you'd like to see deployed, you can submit them on this projects, Github repo issue management. I use 3Scale API Infrastructure to manage access to my APIs, across all parts of the API Evangelist network.
